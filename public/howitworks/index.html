Data Pipeline

[1] Data Collection Using Python Selenium and Beautiful Soup libraries to build web scraping tools, our team collected from the web over 600k unique resumes and 1.1 million salary records representing over 5500 cities across 20 years. Link to Code [2] Data Transformation Data ingestion pipelines were created to standardize the data inputs from various sources. The Models

[1] Document Similarity Model Link to Code We provide career recommendations in a ranked order based on the similarity of a job seeker’s current/past job experiences. Our model is powered by resume information all across the US. This product is a proof of concept and contains a small subset of possible jobs. Our model contains 114 unique job titles. The list of job titles is derived from the availability of salary information.
